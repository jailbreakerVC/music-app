
## Run Locally

Clone the project

```bash
  git clone https://github.com/jailbreakerVC/music-app/
```

Go to the project directory

```bash
  cd music-app/frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack

**Client:** React, DaisyUI

**Server:** Python, Flask
